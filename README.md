
### 项目介绍

vkop 是一个基于 Vulkan 的实现的AI推理引擎，旨在利用GPU计算功能。

### 如何使用

#### 1. 依赖安装
首先需要安装项目的依赖项：
```bash
sudo apt install spirv-tools
```

#### 2. 环境设置
设置 Vulkan ICD 加载器，以 NVIDIA 为例：
```bash
export VK_ICD_FILENAMES=/usr/share/vulkan/icd.d/nvidia_icd.json
```

确保使用正确的 Vulkan 版本：
